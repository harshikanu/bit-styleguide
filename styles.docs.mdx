A simple component that renders an icon. You are required to pass in Props of type and size to choose the icon that you want and the size you want it to display.

# How to Use The Styles

- First, install the package
- Then, for each file you want to use the style in, import the style from the package:

import styles from "@cyber-dive-corp/frontend-styleguide.styles/_styles.module.scss";

# Style Classes & Variables

## Text Style

All text uses _Nunito Sans_ font.

### Font Weight Mixins

These are defined as _@mixins_ only to be reused in the SCSS file.

#### Usage

**`@include normal-weight`**
Normal font weight

**`@include semibold-weight`**
Font weight of 600

**`@include bold-weight`**
Font weight of bold

### Font Line Height Mixins

These are defined as _@mixins_ only to be reused in the SCSS file.

#### Usage

**`@include button-line-height`**
Line height of 1

**`@include heading-line-height`**
Line height of 1.3

**`@include body-line-height`**
Line height of 1.4
